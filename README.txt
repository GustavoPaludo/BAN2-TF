Intruções para a execução da API:

Criando a base de dados:

1-Baixe e instale o Neo4j na máquina.

2-Abra o neo4j.

3-Já com o neo4j aberto, clique em 'New' -> 'Create Project' para criar uma novabase.

4-Clique em 'Add' -> 'Local DBMS'

5-Nomeie a base como desejar e coloque uma senha (O padrão é BAN2 e senha tavo2000), caso deseje colocar uma senha diferente, deve-se mudar isso posteriormente na classe Neo4jConnection.java.

6-Com a base criada, inicie ela clicando em 'Start' e clique em 'Open' para ver a versão web.


Projeto Java:

1-Copie os dois arquivos desta pasta para uma pasta em qualquer local da máquina.

2-Abra esse diretório em qualquer editor com suporte a java (No caso foi utilizado o Eclipse)

3-Na classe Main.java, clique com o botão direito em qualquer local do código e selecione Run as Java Application.


Relatórios:

1-Os arquivos CSV dos relatórios são exportados por padrão para a pasta relatarios localizada em: src/main/java/com/relatorio/relatorios/
